# Number Classifier

## Description
Classifies a number as Small, Medium, or Large based on its value.

## Concepts Used
- Variables
- Conditional statements (if-else)

## How it Works
1. A number is defined in the code.
2. Using if-else conditions:
   - < 10 → Small
   - 10–50 → Medium
   - > 50 → Large
3. Output is printed to the browser.

## How to Run
1. Place the folder inside XAMPP `htdocs`.
2. Open `index.php` via browser at `http://localhost/number-classifier/`.
3. See the classification.
