# Even Odd Checker

## Description
This project checks whether a given number is even or odd.

## Concepts Used
- Variables
- Modulus operator (%)
- Conditional statements (if-else)

## How it Works
1. The number is defined in the code.
2. Using the modulus operator, the program checks if the number is divisible by 2.
3. If divisible, it prints "Even"; otherwise, "Odd".

## How to Run
1. Place the folder inside your XAMPP `htdocs` directory.
2. Open `index.php` in a browser via `http://localhost/even-odd/`.
3. View the result.
