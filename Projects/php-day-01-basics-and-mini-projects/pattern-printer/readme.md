# Pattern Printer

## Description
Prints a numeric pattern in the browser.

Example Output:
1
12
123
1234

## Concepts Used
- Variables
- Loops (nested loops)
- Echo / Print statements

## How it Works
1. Number of rows is defined.
2. Outer loop iterates for rows.
3. Inner loop prints numbers for each row.
4. Pattern is displayed in browser.

## How to Run
1. Place folder inside XAMPP `htdocs`.
2. Open `index.php` via `http://localhost/pattern-printer/`.
3. See the number pattern printed.
