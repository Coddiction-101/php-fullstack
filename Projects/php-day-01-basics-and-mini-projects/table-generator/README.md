# Multiplication Table Generator

## Description
Generates a multiplication table for a given number from 1 to 10.

## Concepts Used
- Variables
- For loops
- Echo / Print statements

## How it Works
1. The number is defined in the code.
2. A `for` loop runs from 1 to 10.
3. Each iteration multiplies the number and prints the result in the browser.

## How to Run
1. Place the folder inside XAMPP `htdocs`.
2. Open `index.php` in browser via `http://localhost/table-generator/`.
3. See the multiplication table.
