# Basic Calculator

## Description
Performs basic arithmetic operations on two numbers: addition, subtraction, multiplication, division.

## Concepts Used
- Variables
- Arithmetic operators (+, -, *, /)
- Echo / Print statements

## How it Works
1. Two numbers are defined in the code.
2. Each arithmetic operation is performed.
3. Result is printed in browser.

## How to Run
1. Place folder inside XAMPP `htdocs`.
2. Open `index.php` at `http://localhost/calculator/`.
3. View all operation results.
