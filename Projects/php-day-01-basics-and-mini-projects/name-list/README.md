# Name List Renderer

## Description
Displays all names stored in an array using loops.

## Concepts Used
- Arrays
- Foreach loops
- Echo / Print statements

## How it Works
1. An array of names is defined.
2. `foreach` loop iterates over the array.
3. Each name is displayed in browser.

## How to Run
1. Place folder inside XAMPP `htdocs`.
2. Open `index.php` via `http://localhost/name-list/`.
3. View the list of names.
